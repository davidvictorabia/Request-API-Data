# Request-API-Data
Pokémon API Data Fetcher (Python)

A simple Python program that connects to an external API to retrieve Pokémon data. The application sends a request to the Pokémon API and displays important information such as the Pokémon's name, ID, height, and weight.

This project demonstrates how to work with REST APIs, handle HTTP requests, and parse JSON data using Python.

🚀 Features

🌐 API Integration
Connects to the public Pokémon API to retrieve Pokémon information.

📡 HTTP Requests
Uses Python's requests library to send GET requests to the API.

📊 JSON Data Processing
Extracts and displays useful data from the API response.

⚡ Dynamic Data Retrieval
Allows users to fetch information about different Pokémon.

Concepts Demonstrated

This project practices important backend and data-handling concepts including:

. API requests

. JSON data parsing

. HTTP response handling

. Functions and modular programming

. Error handling for failed API calls

🛠 Technologies Used

. Python
. Requests
. PokéAPI

📂 Example Output
Name: Pikachu
ID: 25

Learning Purpose: 
This project was created to practice working with APIs and integrating external data sources using Python, which is an important skill for backend development, data engineering, and data-driven applications.
